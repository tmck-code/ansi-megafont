## Font Codepage Mappings

Each font contains 256 characters (0x00-0xFF) mapped to Unicode Private Use Area positions.

| Font Name               | Offset |
|------------------------|--------|
| Topaz Plus A1200       | U+E000 |
| Topaz Plus A500        | U+E100 |
| MoSoul                 | U+E200 |
| MicroKnight Plus       | U+E300 |
| Noodle                 | U+E400 |
| IBM                    | U+E500 |
